Exercício 01 - Variáveis básicas em C#
======================================

**Objetivo**: praticar a declaração de variáveis, atribuição de valores e exibição de dados no console.

**Enunciado**  
Crie um novo projeto de console em C#.  
No `Program.cs`, faça o seguinte:

1. Declare três variáveis:
   - Uma variável `string` para armazenar o seu nome.
   - Uma variável `int` para armazenar a sua idade.
   - Uma variável `double` para armazenar a sua altura em metros (por exemplo, `1.75`).
2. Atribua valores às três variáveis diretamente no código (não precisa ler do teclado).
3. Use `Console.WriteLine` para mostrar cada valor em uma linha separada, no formato:
   - `Nome: <valor do nome>`
   - `Idade: <valor da idade>`
   - `Altura: <valor da altura>`

**Regras**  
- Use apenas o conteúdo de iniciante: declaração de variáveis, tipos básicos (`string`, `int`, `double`) e `Console.WriteLine`.
- Não use laços (`for`, `while`) nem `if`.

**Desafio extra (opcional)**  
Depois de imprimir, altere o valor da idade somando 1 e imprima novamente a idade atualizada.
